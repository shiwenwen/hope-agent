# Hope Agent 0.14.0 — 文件 Diff 与对话阅读体验

> **简体中文** · [English](https://github.com/shiwenwen/hope-agent/blob/v0.14.0/docs/release-notes/v0.14.0.en.md)

> 发布日期：2026-06-29

> 详见 [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.14.0/CHANGELOG.md#0140---2026-06-29)。

v0.14.0 的主线是对**文件改动呈现与长对话阅读体验**的一次集中打磨：文件工具成功后可在对话里直接展开查看 unified diff，右侧 diff 面板补齐了跳转、分栏、折叠、搜索与行号预览等专业能力；完成的对话回合默认自动折叠中间过程、长内容与下挂文件更克制，让长任务的会话更易扫读。配套还带来对话输入历史与快捷指令、更顺手的后台任务面板，以及把长任务运行时超时交还给用户策略统一控制的能力，并对外链 favicon 走统一安全预览链路。

## 亮点

- **文件改动可内联查看 diff**：`write` / `edit` / `apply_patch` 等文件改动工具成功后，在对话里展开工具调用会直接显示固定高度、可滚动的 unified diff，多文件补丁按文件分段展示，无需切到右侧面板即可快速核对改了什么；右侧 diff 面板入口仍保留。
- **Diff 面板补齐专业能力**：查看工具改写文件时会自动定位到首个变更点，支持上一处 / 下一处跳转、统一 / 分栏视图切换、折叠未变更上下文、忽略空白变更、行内高亮、多文件搜索、滚动位置记忆，以及点击行号预览到对应文件行或复制 `path:line` 定位；多文件 diff 的文件头改为吸顶，大 diff 分段渲染避免一次性渲染过多行卡顿。
- **完成回合自动折叠中间消息**：对话回合结束后默认收起前面的思考、工具调用与中间助手消息，只保留「已处理多久 / 多少条消息」摘要和最终回复，减少长任务里的信息干扰与渲染节点数量；摘要可随时展开查看细节，历史会话里的旧消息也按同样规则折叠。设置页「对话与上下文」新增开关可关闭该行为。
- **对话输入历史与快捷指令**：对话输入框空草稿时可用 `↑` / `↓` 浏览当前会话最近输入；用户消息底部工具栏新增「添加到快捷指令」，保存后可在输入框通过 `#` 唤起快捷指令菜单快速插入，placeholder 也补充了 `@` / `#` 的友好提示。
- **运行时超时改为用户策略驱动**：长任务的运行时超时（定时任务运行 / MCP 调用 / ACP 回合 / 异步任务等）现在由用户策略统一控制，可显式设为「不限」（`0` = 无超时），避免模型自行设的短超时把本应稳定长跑的任务中途掐断；新增「模型运行时超时覆盖」策略可选择允许 / 仅审计 / 忽略模型提供的超时值。用户设置的正向超时预算仍是硬上限——模型覆盖只能收紧、不能放宽。

## 文件改动与编辑可视化

- **文件编辑行数计数更直观**：文件编辑工具、工作台与 diff 面板里的 `+N/-M` 行数统计改为计数牌滚动动效，并在 `edit` / `apply_patch` 执行中先显示可安全推断的预估行数、完成后用真实 diff 结果校准；编辑工具行的目标文件名前也会显示文件类型图标，更容易快速扫清正在改哪个文件。编辑失败或无操作时不再误显行数变更（完成后必须使用后端真实 diff metadata）。
- **文件工具调用目标显示更清爽**：对话里的 `read` / `write` / `edit` / `apply_patch` 等文件读写工具不再直接展示完整路径，改为文件类型图标 + 文件名；悬浮文件名仍可查看完整路径，多文件补丁会在悬浮提示中列出目标路径。
- **工作台 Git 变更可直接查看 diff**：点击工作台「环境」里的「变更」行会打开右侧 diff 面板，按文件纵向展示当前会话工作目录内的 Git 改动并支持逐文件收起 / 展开；server 模式下该接口也收紧到会话工作目录范围，符号链接按 Git 链接文本展示，不会跟随读取工作区外文件。

## 对话阅读体验

- **聊天长内容与下挂文件更克制**：超长用户消息默认折叠为带渐变遮罩的预览，可展开查看全文；助手回复底部生成 / 修改文件较多时默认展示前 6 个并可展开。完成回合自动折叠时，工具过程中生成的文件会提升到最终回复底部，不再藏在「已处理」细节里。
- **查看子智能体对话不再切走主对话**：除左侧对话列表外，消息气泡、团队面板、后台任务面板与工作台里的子智能体对话入口现在统一打开实时弹层查看并复用主对话消息流能力；在消息对话页查看子智能体时不会再把当前主 Agent 会话切走。

## 后台任务

- **后台任务面板跟随任务状态更顺手**：触发后台任务后会自动展开右侧后台任务面板；运行中的实时输出会智能跟随滚到底部，并在用户手动翻看历史输出时保留位置、提供回到底部入口；每个后台任务条目可单独展开 / 收起，避免长输出把整块面板撑满。
- **工作台后台任务可直接展开操作**：工作台里的后台任务区块现在复用后台任务面板同一套任务行能力，可直接展开查看实时输出 / 结果预览、取消仍在运行的任务，并与独立后台任务面板共享展开状态；工作台仍保留紧凑展示和「查看全部」入口。

## 其它改进

- **Responses / Codex 工具轮不再反复讲同一句进度**：工具调用轮现在会把已流出的助手进度文本写回模型可见历史，工具后的下一次采样能接上前文；自然结束时只持久化最终回复，用户停止时也保留已经看到的部分回复，避免后续上下文丢段或重复。
- **侧边栏搜索召回更完整且更快**：全局会话搜索现在能命中会话标题，并用 FTS5 trigram 索引补足正文子串匹配（如单词片段 / 常见中英文片段），不再依赖 `%LIKE%` 扫描整张消息表；标题命中会直接打开会话，避免跳转到不存在的消息气泡。
- **后端直发通知跟随界面语言**：Rust 后端现在能读取应用语言偏好，用于 IM 渠道启动恢复 / 会话接管 / 工具审批 / ask_user 交互、定时任务失败投递包装等直接外发文案，不再固定英文；同时保留前端 UI 事件的 key + fallback 渲染边界，避免把模型语言偏好误用成产品界面语言。
- **进度更新设置文案更准确**：设置页「对话与上下文」里的工具调用说明开关改为「进度更新」口径，说明它控制关键步骤的简短进度与回合总结；12 种界面语言已同步更新，并移除针对特定模型的特殊开启建议。

## 安全

- **外链 favicon 走统一安全预览链路**：Markdown 网页链接、URL 预览卡片与工作台来源现在可显示站点 favicon，但图标统一由后端经过 SSRF 校验、重定向逐跳校验、大小限制与图片类型校验后转为 data URL；页面声明的 favicon 会优先保留，失败再回退 `/favicon.ico`。Markdown 链接的 favicon 获取改为悬停 / 聚焦后触发，并带缓存、并发与数量上限，避免历史消息一次渲染大量外链时打出无界后端请求。
