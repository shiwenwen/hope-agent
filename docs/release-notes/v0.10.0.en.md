# Hope Agent 0.10.0 — Knowledge Base: Your Second Brain

> [简体中文](https://github.com/shiwenwen/hope-agent/blob/v0.10.0/docs/release-notes/v0.10.0.md) · **English**

> Released: 2026-06-12

> See [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.10.0/CHANGELOG.md#0100---2026-06-12) for details.

v0.10.0 introduces a whole new subsystem — the **Knowledge Base** — that turns Hope Agent from a chat assistant into your second brain: local-first note management, AI search and rewriting across notes, a "sprite" that surfaces ideas as you think, plus quality-of-life upgrades like `@`-mentioning skills right from the input box.

## Highlights

- **Knowledge Base: your second brain**: a brand-new local knowledge-management subsystem. Notes are real `.md` files on disk (the single source of truth); you can mount a local folder or an external vault (e.g. Obsidian) with live two-way sync. Capture, organize, link (wikilinks), tag, backlinks and a knowledge graph are all built in, and search fuses keyword and semantic retrieval. Sessions and projects attach knowledge bases on demand with read-only / read-write permissions, and incognito sessions get zero access.
- **Built-in AI chat panel + selection rewrite**: a new "AI chat" panel on the right of the Knowledge Base lets you talk with the current note, have the AI search across notes and write / rewrite them, switch agents, and pull in more notes for combined answers — with per-document chat history preserved. Select text in the editor to add it to the chat as a quote, or use the floating "Quick rewrite" bar to have the AI rewrite the selection directly (preview the diff, then apply).
- **Sprite / inspiration mode (off by default)**: when enabled, pausing while editing a note lets the sprite combine the current document, your recent edits, the conversation, memory and cross-session awareness to surface a light suggestion bubble (writing tips / feedback / connections / encouragement) in the chat panel — tap "Respond" to keep the thread going. Multiple triggers are supported (opening a note / after a reply / sustained writing / large paste), throttled with a cooldown and an hourly cap, and never firing in incognito.
- **Ready out of the box**: a fresh install creates a built-in knowledge base on first launch (with a localized welcome note), so you can start capturing with zero setup; delete it and it won't be recreated. A new AI-facing "Knowledge Base methodology" skill also guides the model to capture, organize and maintain your notes more effectively.
- **`@`-mention built-in skills**: type `@` in the input box to attach the Word / PPT / Excel trio, browser control, and Mac control (macOS only) as skills for the current conversation — attach several at once, set apart from files and notes by a distinct rose-pink color.

## Other improvements

- **Multilingual recap output**: the AI sections, headings and summary of `/recap` deep recaps can follow your interface language, or you can pick a separate output language under Settings → Recap.
- **Keep the computer awake**: a new toggle under Settings → General → System keeps the machine awake so long tasks, downloads and background activity aren't interrupted by system sleep (the screen can still turn off).
- **Refined session sidebar**: the Agents block moved above Projects, and the Agents / Projects / Sessions category headers stick to the top while you scroll up.
- **Codex default model upgraded to GPT-5.5**: new logins default to GPT-5.5, and existing users see it pinned to the top of their model list after upgrading.
