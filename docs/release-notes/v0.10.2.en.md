# Hope Agent 0.10.2 — Layout-Faithful Office Document Preview

> [简体中文](https://github.com/shiwenwen/hope-agent/blob/v0.10.2/docs/release-notes/v0.10.2.md) · **English**

> Released: 2026-06-13

> See [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.10.2/CHANGELOG.md#0102---2026-06-13) for details.

v0.10.2 is a patch for v0.10.0 that restores layout-faithful preview for Office documents.

## Added

- **Layout-faithful Office document preview**: when previewing Word / Excel / PowerPoint (`.docx` / `.xlsx` / `.pptx`), the right-side panel now renders close to the original layout — Word document formatting, Excel tables with merged cells, and PowerPoint slide paging, all with zoom / fit-to-width — instead of plain extracted text. Files that can't be rendered (legacy `.doc` / `.ppt`, corrupted, or oversized) fall back to text extraction automatically.
