# Hope Agent 0.10.2 — Office 文档预览还原版式

> **简体中文** · [English](https://github.com/shiwenwen/hope-agent/blob/v0.10.2/docs/release-notes/v0.10.2.en.md)

> 发布日期：2026-06-13

> 详见 [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.10.2/CHANGELOG.md#0102---2026-06-13)。

v0.10.2 是 v0.10.0 的补丁版，补回了 Office 文档的版式化预览能力。

## 新增

- **Office 文档预览还原版式**：预览 Word / Excel / PowerPoint（.docx / .xlsx / .pptx）时，右侧面板直接按接近原文件的版式渲染——Word 文档排版、Excel 表格与合并单元格、PPT 幻灯片翻页，均可缩放 / 适应宽度，不再只是纯文本提取；无法渲染的文件（旧 `.doc` / `.ppt`、损坏或过大）会自动回退到文本提取。
